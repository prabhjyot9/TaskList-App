## Running the App

To run this app locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/task-list-app.git
   cd task-list-app
   ```
2. **Install dependencies**:
```bash
   npm install
   ```
3. **Start the application:**:
```bash
   npm run start
   ```
